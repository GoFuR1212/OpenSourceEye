## 🚀 Установка и запуск

### Способ 1: Готовый .exe файл (рекомендуется)

1. Перейдите в раздел **[Releases](../../releases/latest)**
2. Скачайте файл `OpenSourceEye.exe`
3. Запустите двойным щелчком мыши
4. Следуйте инструкциям в терминале

> **Примечание:** Windows может показать предупреждение "Защитник Windows предотвратил запуск неизвестного приложения". Нажмите **"Подробнее"** → **"Выполнить в любом случае"**. Это связано с отсутствием цифровой подписи, а не с вредоносностью.

### Способ 2: Запуск из исходного кода (для разработчиков)

```bash
# Клонируйте репозиторий
git clone https://github.com/GoFuR1212/OpenSourceEye.git

# Перейдите в папку
cd OpenSourceEye

# Запустите .exe
OSINT > OpenSourceEye > main.exe
